 
 
Sistema operativo
Es el software base que controla los recursos del equipo.
Permite que funcionen todos los programas y hace posible la comunicación.
 
Node.js
Es un entorno para ejecutar código JavaScript fuera del navegador.
Se usa para crear servidores y aplicaciones web de forma rápida.
 
Aplicación server.js
Es el archivo donde está la lógica principal del servidor.
Aquí se definen las rutas y las respuestas a las solicitudes.
 
Puerto
Es un número que funciona como una puerta de comunicación.
Indica a qué servicio deben llegar las peticiones que se envían.
 
Logs
Son archivos que registran todo lo que pasa en el servidor.
Ayudan a ver su funcionamiento y a detectar errores.
